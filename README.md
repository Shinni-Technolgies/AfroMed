## Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-org/afromed.git
cd afromed
```

2. Install packages

```bash
npm install --legacy-peer-deps
```

3. Copy environment file

```bash
cp .env.example .env
```

4. Run project

```bash
npm run dev
```

## Why AfroMed?

AfroMed is purpose-built for healthcare professionals and institutions:

- React 19 with TypeScript for type-safe development.
- Material UI v7 components for a polished UI.
- Fully responsive across all modern browsers.
- Medical-focused navigation: Patients, Appointments, Pharmacy, Lab, Billing.
- Clean, scalable code structure.
- Vite-powered for fast development and builds.

## Browser Support

Chrome, Edge, Safari, Firefox, Opera — all modern browsers supported.

## Technology Stack

- React 19 with TypeScript
- Material UI v7 components
- React Hooks API & Context API for state management
- SWR for data fetching
- React Router v7 for navigation
- Vite for fast builds
- Formik + Yup for forms
- MUI X Charts for data visualization
- Ant Design Icons
